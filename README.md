# OpenAI Key Checker

A minimalistic dark-themed web service to check if your OpenAI API key is expired or valid.

## Features

- Simple, clean interface with dark theme
- Real-time key validation
- Clear error messages for expired, invalid, or rate-limited keys
- Secure server-side validation

## Usage

Enter your OpenAI API key (starting with `sk-`) and click "Check Key" to verify if it's valid and working.

## Local Testing

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open `http://localhost:5500` in your browser.

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Serverless Functions
- OpenAI API
